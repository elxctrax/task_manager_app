1. Clone repo and enter folder:
   git clone https://github.com/elxctrax/task_manager_app.git
   cd task_manager_app

2. Install dependencies:
   composer install

3. Copy env file and set DB:
   cp .env.example .env
   (edit .env with your DB name, user, and password)

4. Generate key and migrate:
   php artisan key:generate
   php artisan migrate

5. Run app:
   php artisan serve
   Open http://127.0.0.1:8000
